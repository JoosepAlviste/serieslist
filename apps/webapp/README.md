# Serieslist webapp

The front-end for the Serieslist web application.


## Code checks

### ESLint

```sh
pnpm lint
```

### TypeScript

```sh
pnpm tsc
```

### Tests

```sh
pnpm test
```


## Development

### GraphQL Code Generator

To generate TypeScript types for GraphQL requests, run:

```sh
pnpm codegen
```
